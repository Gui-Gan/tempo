Tempo

Tempo is a program for pulsar timing data analysis.

Download

Tempo has now been imported into Sourceforge. The CVS version can be checked out out via:

cvs -z3 -d:pserver:anonymous@tempo.cvs.sourceforge.net:/cvsroot/tempo co -P tempo

Or if you have a Sourceforge account:

cvs -z3 -d USERNAME@tempo.cvs.sourceforge.net:/cvsroot/tempo co -P tempo
